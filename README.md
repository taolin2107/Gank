# Gank

Gank是基于[干货集中营（http://gank.io）](http://gank.io)的开放api开发的一款非官方客户端，致力于提供精选的Android、iOS、Web等类别的技术干货。

项目采用MVP架构，结合Dagger2, RxJava2, RxLifeCycle, Retrofit等优秀开源框架打造的一个流行的app架构，以及全部使用Kotlin开发。

## Features

 * 采用主流的Android app开发架构。
 * 界面简洁、操作流畅。
 * 抽屉式菜单，分类浏览。
 * 下拉刷新，底部上拉加载更多。
 * 图片卡片式布局。
 * 图片自动缓存。
 * 基于github实现的版本更新。
 * 支持用户反馈。
